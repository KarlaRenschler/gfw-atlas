--- layout: default title: "About the project" subtitle: "MY Forest Watch is a powerful tool for Governments, NGOs and the private sector to build a living, dynamic forest monitoring system that provides unbiased and up-to-date information on forest dynamics within their focus areas. Build on ArcGIS online and powered by the Global Forest Watch API this platform allows everyone to build their own localized version of Global Forest Watch using their own data layers and integrate within their own website. Explore the gallery to see how others use this platform and watch the tutorial videos to learn how to build your own." ---
