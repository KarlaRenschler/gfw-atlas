About the platform

My Forest Atlas is a powerful tool for a living, dynamic forest monitoring system that provides unbiased and up-to-date information on the DRC's forest sector. Built on a geographic information system (GIS) platform, the Atlas aims to strengthen forest management and land use planning by bringing information on all major land use categories onto the same standardized platform.
